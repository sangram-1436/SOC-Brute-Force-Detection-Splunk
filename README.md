# SOC Alert Triage, Brute Force Detection & Incident Escalation (Splunk)

## Project Overview
This project demonstrates an end-to-end SOC workflow using Splunk SIEM to detect, investigate, and respond to brute-force authentication attacks on Windows systems.

## Objectives
- Detect brute-force and password spray attacks
- Perform alert triage and investigation
- Classify incident severity
- Document escalation and remediation steps

## Tools Used
- Splunk Enterprise
- Windows Security Event Logs (Event ID 4625)

## Detection Logic
- Threshold-based brute-force detection
- Password spray detection using unique user count
- Scheduled alerts in Splunk

## Incident Handling
- Incident identification and documentation
- Attacker IP investigation
- Severity classification and escalation
- Incident reporting with recommendations

## Skills Demonstrated
- SIEM log analysis
- Alert triage
- Incident response
- SOC documentation
